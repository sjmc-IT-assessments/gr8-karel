  • Return to starting position
  • End facing East

Guide:

  • Create a function to turn around and come back
  • Make sure to put a ball on the last square too

2. Opposite Squares (5 marks)

Start World:

End World:

Requirements:

  • World is at least 4 columns wide and 2 rows high
  • Draw a square at both ends of the world
  • Each square should be outlined with balls
  • End facing East at bottom right corner

Guide:

  • Create a function to build one square using a loop
  • Use move() and turnLeft() to draw square sides
  • Move to the other end of the world for second square
